Programming Experience

Voor deze opleiding had ik minimale ervaring met programmeren. zo heb ik voorgaand een game proberen te maken met GameMaker Studio2, ook heb ik 2 jaar terug een simpele website proberen te bouwen, echter was dit een groepsopdracht. Er werd alleen gebruikgemaakt van de taal html, het proces werd stapsgewijs gevolgd via een youtube tutorial die maar een paar minuten duurde. Eigenlijk kun je deze showcase website dus als mijn eerste echte ICT project beschouwen.
